# 电影/动漫推荐墙

一个精美的电影和动漫推荐展示网站，展示精选的优质影视作品。

## 功能特点

- 🎬 展示8部精选电影和动漫作品
- 📱 响应式设计，适配各种设备
- ✨ 精美的卡片悬停效果
- 🔍 分类筛选功能（全部/电影/动漫/高分）
- 🎨 现代化的视觉设计
